# Diabetes Prediction Using SVM

## Project Overview
This project predicts whether a person is diabetic using Support Vector Machine (SVM).

## Features
- Data Cleaning & EDA
- Outlier Handling
- Hyperparameter Tuning
- ROC-AUC Evaluation
- Flask Web Application

## Model Performance
- AUC Score: 0.81
- Recall (Diabetic): 75%
- Accuracy: 71%

## Technologies Used
- Python
- Scikit-Learn
- Flask
- HTML/CSS

## How to Run
1. Clone the repo
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py
